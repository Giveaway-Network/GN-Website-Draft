About the Giveaway Network

The Giveaway Network (GN) is an independant driven plateform founded in 2017 and open source. Originaly located on Discord, it provides a free way to host games giveaways. Those games giveaways are run by "giveaway communities", also hosted on the website. The main interest is to get games for free.
